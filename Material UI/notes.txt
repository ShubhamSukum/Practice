Create a material UI app with typescript
=> npx create-react-app appname --template typescript

To install material UI in your application
=> npm install @mui/material @emotion/react @emotion/styled 

Add a default bottom margin
=> <Typography variant="h4" gutterBottom>H4 Heading</Typography>

How to install material UI's ICON ?
=> npm install @mui/icons-material

aria-label is like alt in img (it is used in symentic purpose)

We can use Stack,Box from MaterialUI instead of using DIV.
